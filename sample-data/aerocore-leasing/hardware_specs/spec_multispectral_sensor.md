# MultiSpectral-CropSensor Pro — Specifications
## Model Number: MSCP-11029-A

*   **Bands Captured:** Red, Green, Blue, Red Edge, Near-Infrared (NIR)
*   **Resolution:** 5.2 Megapixels per band
*   **Field of View (FOV):** 64 degrees
*   **Calibration:** Automated downwelling light sensor (DLS) adjusts for sun intensity
*   **Trigger Mode:** Automated timer (every 1.5 seconds) or GPS-coordinate grid trigger.
